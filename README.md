# Java Zadaci

Ovaj repozitorij sadrži osnovne zadatke iz Jave:

## 📂 Struktura

- `funkcije/` → primjeri funkcija (obična, void, rekurzivna)
- `stringovi/` → manipulacija stringova
- `nizovi/` → rad sa nizovima

## ▶️ Pokretanje

Svaki folder je poseban Java projekat.

1. Otvoriti folder (npr. `stringovi`) u VS Code ili IntelliJ
2. Pokrenuti `Main.java`

## 📝 Napomena

Svaki projekat ima svoj `main` metod i može se pokretati nezavisno.
